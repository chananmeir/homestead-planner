🧹 Clear Plant Icon Preference

This tool clears the old icon preference setting that may be preventing PNG icons from displaying correctly in Homestead Planner.

⚠️ Important: Make sure you're viewing this page from http://localhost:3000 (or your frontend server) for this to work correctly.
1 Click the button below to clear the preference:
2 After clearing, return to your Homestead Planner tab and refresh the page.

Manual Method (Alternative)

If the button doesn't work, you can manually clear it:

  1. Open your Homestead Planner tab (http://localhost:3000)
  2. Open Browser DevTools (press F12)
  3. Go to the Console tab
  4. Type: localStorage.removeItem('plantIcon_displayMode')
  5. Press Enter
  6. Refresh the page
