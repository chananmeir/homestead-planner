## Understanding "Plant" vs. "Per Square Foot"

**Important Distinction:**

For crops planted individually (tomatoes, peppers, cabbage):
- **lbs/plant** = yield from one plant

For crops planted densely (carrots, beets, radish, greens):
- **lbs/plant** = yield from ONE root/head
- **lbs/sqft** = yield from entire square foot area
- Example: Carrots - 0.15 lbs per carrot, but 16 carrots per sqft = 2.4 lbs/sqft

### Your System Handles This Correctly

The Garden Season Planner uses:
- **Individual plants method**: tomato, lettuce, pepper → uses lbs/plant
- **Seed density method** (MIGardener): carrots, radish, greens → uses lbs/sqft

So the carrot update (1.5 → 0.15 lbs) is correct because:
- If you plant 12 carrot "plants" (seeds), you get 12 carrots
- 12 carrots × 0.15 lbs = 1.8 lbs total ✓
